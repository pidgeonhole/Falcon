Basic usage

After selecting the language you are working with, you can start typing away in the editor. Otherwise, you can simply copy and paste your code from your favourite IDE and submit it.

The number of test cases and the inputs for the test cases will be read from the standard input. After cleaning the standard input, the answer you give must then be printed to the standard output. Don't worry if you don't understand what we're saying now. We'll go through some examples in this document.

Simple Example: Summation

Assume we have the following problem where we are given some list of numbers and the question is: "What is their sum?"

Usually, we will start off by telling you the expected input you'd receive and go on to tell you what is the output of the answer we expect. The game can contain X number of test cases and in each test case, contain Y number of trials. You won't need to worry about the number of test cases. However, you'll have to take care of the number of trials if we specify them. For example, in this case, we have 1 test case and 3 trials. Therefore, the input could look like this: 

                
                    3             # number of trials in 1 test case
                    1 2 3 4 5 6   # input 1
                    2 3 4 5 6     # input 2
                    6 5 -1 2 2    # input 3

All your output needs to be printed to standard output (described further below). Thus, returning to the example, the correct answer will have an output that looks like this:

                
                    21
                    20
                    14

Reading and writing to standard input/output

We'll use the example "summation" question above in this tutorial. Here's how to solve it.

  • R 
                            
                            N = int(input()) # Read from standard input with input() and convert to integer
                            
                            for i in range(N):
                                string = input()  # Now I've read in the numbers, i.e. 1 2 3 4 5 6 but it's a string
                                nums = [int(j) for j in string.split()]   # converts string to list of integers
                                answer = sum(nums)  # Get answer
                                
                                print(answer)  # print your final answer to standard output
                        
                        
                            
                            f = file("stdin")
                            lines = readLines(f)  # Read all lines from standard input

                            N = as.numeric(lines[1])  # First input is number of test cases
                            test.cases = lines[-1]        # Subsequent inputs 

                            for (i in seq(N)){
                                # Split each sequence by spaces 
                                list.of.char = strsplit(test.cases[i], '\\s+')[[1]]
                                
                                # Change from char vector to numeric vector
                                list.of.num = as.numeric(list.of.char)
                                
                                # Prepare answer
                                answer = sum(list.of.num)

                                # Print answer to standard output
                                write(answer, stdout())
                            }

Essentially, printing to standard output is as easy as print() in Python. There are equivalents in R and the other languages so we won't go through all of them here. However, one last thing to note is that you should only print your final answer because we register that as your intended solution.
