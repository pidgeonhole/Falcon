Basic usage
After selecting the language you are working with, you can start typing away in the editor. Otherwise, you can simply copy and paste your code from your favourite IDE and submit it.

The number of test cases and the inputs for the test cases will be read from the standard input. After cleaning the standard input, the answer you give must then be printed to the standard output. Don't worry if you don't understand what we're saying now. We'll go through some examples in this document.
Simple Example: Summation
Assume we have the following problem where we are given some list of numbers and the question is: "What is their sum?"
The input file could look like the following 
                
                    3             # number of test cases
                    1 2 3 4 5 6   # input 1
                    2 3 4 5 6     # input 2
                    6 5 -1 2 2    # input 3

Ideally, your answer (output) should be printed like so 
                
                    Case #1: 21
                    Case #2: 20
                    Case #3: 14
Reading and writing to standard input/output
We'll use the example "summation" question above in this tutorial. Here's how to solve it.

                            f = file("stdin")

                            N = as.numeric(lines[1])  # First input is number of test cases
                            test.cases = lines[-1]        # Subsequent inputs 

                            for (i in seq(N)){
                                # Split each sequence by spaces 
                                list.of.char = strsplit(test.cases[i], '\\s+')[[1]]
                                
                                # Change from char vector to numeric vector
                                list.of.num = as.numeric(list.of.char)
                                
                                # Prepare answer
                                answer = sum(list_of_num)
                                message = sprintf("Case #%d: %d", i, answer)
                                
                                # Print answer to standard output
                                write(message, stdout())
                            }
